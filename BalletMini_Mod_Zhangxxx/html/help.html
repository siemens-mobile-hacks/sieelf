BalletMini 安装说明

（SGOLD 4:\改为0:\）

1. BalletMini.elf 复制到文件夹4:\ZBin\BalletMini \

2. 创建文件夹

4:\ZBin\BalletMini\OmsCache

4:\ZBin\BalletMini\UrlCache

4:\ZBin\BalletMini\Bookmarks

3. 运行OperaMini, go to Menu-> Settings-> Settings-> code writer-> Options-> Export（此步为导出，然后把文件重命名为AuthCode）,AuthCode放入文件夹4:\ZBin\BalletMini\

4. 放置图片 OMS_SMALL.png, OMS_BIG.png, URL_SMALL.png 和 URL_BIG.png(以上好象会自动生成)，图片floder.png,bookmark.png在4:\ZBin\img\

5. 在 4:\ZBin\etc\extension.cfg 写入:

 

[oms]

RUN=4:\ZBin\BalletMini\BalletMini.elf

SMALL=4:\ZBin\img\OMS_SMALL.png

BIG=4:\ZBin\img\OMS_BIG.png

 

[url]

RUN=4:\ZBin\BalletMini\BalletMini.elf

SMALL=4:\ZBin\img\URL_SMALL.png

BIG=4:\ZBin\img\URL_BIG.png

6. 书签

   书签的建立：可以写网址到一个TXT文件然后重命名为.url放到 4:\ZBin\BalletMini\Bookmarks作为书签

   书签文件夹：在4:\ZBin\BalletMini\Bookmarks下建立文件夹就是书签的文件夹

7. 搜索引擎的使用

   如GOOGLE：保存http://www.google.cn/search?hl=zh-CN&q=为google.url到Search文件夹

8.模板的使用

   编辑模板文字，保存为Templates.txt编码为GB2312(ANSI)

9. 按键

   左软键：菜单

   右软键：后退

   3/9：页首/页尾

   0：查看当前页面文字
